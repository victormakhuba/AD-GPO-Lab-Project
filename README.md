## Active Directory & Group Policy Home Lab

### Overview
Built a Windows Server domain environment with Active Directory, 
organizational units, group policies, and role-based access control.

### Environment
- Windows Server 2019 (Domain Controller)
- Windows 10 Client
- Active Directory, DNS
- Group Policy Management

### Key Implementations
- OU design by department and device type
- GPO scoping and security filtering
- NTFS permissions and SMB shares
- Least privilege access enforcement
- User and computer policy separation

### Validation
- RSOP used to confirm applied policies
- Access tested with standard user accounts
